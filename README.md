# 🎮 坦克大战 (Tank Battle)

经典FC风格坦克大战游戏，纯HTML5 Canvas实现，无需任何依赖，双击即可运行！

## 🎯 游戏特色

- 🎨 经典NES风格像素画面
- 🗺️ 3个精心设计的关卡地图
- 🧱 可破坏的砖墙 & 不可破坏的钢铁墙
- 💧 水域障碍（坦克无法通过）
- 🌲 森林隐藏效果
- ⭐ 基地保护机制
- 💥 爆炸特效
- 📊 得分系统 & 生命系统
- 📱 支持移动端触摸操控
- ⏸️ 暂停功能

## 🕹️ 操作方式

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → 或 W S A D | 移动坦克 |
| 空格键 | 发射炮弹 |
| P | 暂停/继续 |

### 移动端
- 触摸滑动控制方向
- 自动射击

## 🚀 快速开始

1. 直接双击 index.html 在浏览器中打开
2. 点击「开始游戏」按钮
3. 或者克隆仓库后用任意HTTP服务器运行：

`ash
git clone https://github.com/<你的用户名>/tank-battle.git
cd tank-battle
python -m http.server 8000
`

## 🎮 游戏规则

- 消灭关卡中的所有敌方坦克即可过关
- 保护地图底部的⭐基地，基地被摧毁则游戏结束
- 每消灭一个敌人 +100分，通关 +500分
- 共有3条生命，生命耗尽游戏结束
- 关卡会循环，每关敌人数量递增
